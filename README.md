⚙️ Installation
Clone the repository:
git clone https://github.com/your-username/jarvis-ai.git
cd jarvis-ai
Create virtual environment:
python -m venv env
env\Scripts\activate   # Windows
Install dependencies:
pip install -r requirements.txt
▶️ Usage
python main.py

🎤 Start speaking commands and Jarvis will respond!

🔑 Configuration
Add your API keys in .env or config file
Customize commands inside commands/ folder
